This plugin show automatically your title, description and you post image in facebook when you add a link in the time line. Not needs configuration.

== Description ==

This plugin it is completly automatic, the plugin take the information for the current post and create the metadata for facebook, title, description, url, image, site title.
The image that it is showing for facebook, it is the featured image of the post.
When you add your url to facebook, it will show the right information in facebook, without any configuration. 
If you use woocommerce it will works for all your products of course.

== Installation ==

The best way it is from your WordPress dashboard

    1- Visit 'Plugins > Add New'
    2- Search for ' simple automatic add link facebook'
    3- Activate  simple automatic add link facebook from your Plugins page.


From WordPress.org 

    1- Download  simple automatic add link facebook.
    2- Upload the ' simple automatic add link facebook' directory to your '/wp-content/plugins/' directory, using your favorite method (ftp, sftp, scp, etc...)
    3- Activate  simple automatic add link facebook from your Plugins page. 
    
After the plugin it is activate, you can start to share your links in facebook.
